# 🚀 TaskFlow — Smart Task Management System

TaskFlow is a web-based task management application designed to help users efficiently organize, prioritize, and track their daily tasks. It provides smart scheduling, real-time search, filtering, and persistent storage using Local Storage.

---

## 🌐 Live Application

👉 https://manyasohan.github.io/TaskFlow/

---

## ✨ Features

- ➕ Add, delete, and mark tasks as complete  
- 🎯 Priority levels: High, Medium, Low  
- 🧠 Smart scheduling based on priority  
- 🔍 Real-time search functionality  
- 📂 Filter tasks (All / Active / Completed)  
- 📅 Due date tracking  
- 📊 Task counter display  
- 💾 Data stored using Local Storage  
- 🎨 Clean and responsive user interface  

---

## 🛠️ Technologies Used

- HTML — Structure of the application  
- CSS — Styling and layout  
- JavaScript — Functionality and logic  

---

## 🧠 How It Works

- Tasks are stored as JavaScript objects with properties like text, priority, completion status, and due date  
- DOM manipulation is used to dynamically update the UI  
- Local Storage is used to persist data across sessions  
- Tasks can be filtered, searched, and sorted based on priority
